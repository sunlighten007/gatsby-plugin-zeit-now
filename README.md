# gatsby-plugin-now

Deploy GatsbyJS site on Zeit Now.

## Install

With Yarn:

```bash
yarn add @universse/gatsby-plugin-now
```

Or with npm:

```bash
npm install --save @universse/gatsby-plugin-now
```

## Usage

Add plugin to gatsby-config.js

```js
plugins: ['@universse/gatsby-plugin-now']
```
