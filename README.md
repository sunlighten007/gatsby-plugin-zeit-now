# gatsby-plugin-zeit-now

Deploy GatsbyJS site on Zeit Now.

## Install

With Yarn:

```bash
yarn add gatsby-plugin-zeit-now
```

Or with npm:

```bash
npm install --save gatsby-plugin-zeit-now
```

## Usage

Add plugin to gatsby-config.js

```js
plugins: ['gatsby-plugin-zeit-now']
```
